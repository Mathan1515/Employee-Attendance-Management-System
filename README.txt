🧾 Employee Attendance Management System
A complete web-based Employee Attendance Management System built with PHP, MySQL, HTML/CSS, and JavaScript. This system allows organizations to efficiently track employee attendance, view records, generate reports, and manage employee data.

✨ Features
📅 Daily attendance marking (Present / Absent / Half Day)

👤 Employee registration and management

⏰ In-time and Out-time recording

📊 Attendance reports by date and employee

🔒 Secure login for Admin & Employees

📥 QR code-based attendance option (optional feature)

📌 Easy to deploy for startups and small companies

💻 Tech Stack
Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: Core PHP

Database: MySQL

QR Integration: html5-qrcode (for contactless scan-based attendance)
